流式渲染 (Streaming) Demo

服务端分块发送，前端逐步渲染

1. 流式文本

模拟逐句返回，每句到达后立即追加到页面。

2. 流式 JSON (NDJSON)

每行一个 JSON，收到一行就渲染一条。

    3. 流式 HTML 片段

    服务端推送 HTML 块，前端直接插入 DOM。

    4. ReadableStream 逐字输出

    使用 Fetch + ReadableStream 逐字显示。

    5. 混合类型流（含 meta + md）

    同一流里按 type 区分：meta 显示为标题，md 流式渲染为 Markdown。

    6. SSE（EventSource）

    使用 EventSource 订阅，Content-Type: text/event-stream，标准服务端推送协议。

    7. Fetch + SSE 格式

    同样请求 text/event-stream 接口，用 fetch + ReadableStream 读流，前端按 SSE 格式（data: ...\n\n）解析后渲染。
